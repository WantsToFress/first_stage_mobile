# first_stage_mobile
To deploy run
* npm i -g npm
* npm i -g react-native-cli
* git clone <folder>
* cd <folder>
* npm i
* react-native link
* react-native run-android (run-ios)
  
If using Android/ before running create file 'local.properties' inside android/ folder with string sdk.dir=$PATH_TO_ANDROID_SDK
